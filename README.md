# Ripple Paper Wallet
Paper wallet for Ripple (XRP)

## Validate checksum
Checksum (SHA1) must be `2f64d541e0a16873e2b5e9ba5cb5d566f215a27c`;

On OSX you can easily check this by running `sha1sum index.html` in the terminal.
